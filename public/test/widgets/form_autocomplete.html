Adding NCBO Widgets to Your Site

Form Autocomplete

Fill in a field in this form with a concept from the NCI Thesaurus [ontologyId=NCIT]

  • Example 1 (start typing a class name to get a full URI)
    <input type="text" name="a" class="bp_form_complete-NCIT-uri" />
     
  • Example 2 (get the class ID)
    <input type="text" name="b" class="bp_form_complete-NCIT-shortid" />
     
  • Example 3 (get the class preferred name)
    <input type="text" name="c" class="bp_form_complete-NCIT-name" />
     

Optional attributes

  • Include class definitions. Use data-bp_include_definitions="true". This will display definitions with the list of classes that are returned. For example, use this code to display search results for 'heart' concepts, with definitions, in the NCI Thesaurus.
    <input type="text" name="d" data-bp_include_definitions="true" class="bp_form_complete-NCIT-name" /> Try it:
     
  • Limit lookup to classes below a root class. Use data-bp_search_branch="{conceptid}" (replace {conceptid} with the class id for the class to use as the branch root). For example, use this code to limit the search for 'heart' to classes within the 'Anatomic_Structure_System_or_Substance' branch of the NCI Thesaurus.
    <input type="text" name="d" data-bp_search_branch="Anatomic_Structure_System_or_Substance" class="bp_form_complete-NCIT-name" /> Try it:
     
  • Object types
     
    • Search 'class' and 'property' objects. Use data-bp_objecttypes="class,property". For example, use this code to search for 'function' among both 'class' and 'property' objects from the "Adverse Event Reporting Ontology".
      <input type="text" name="d" data-bp_objecttypes="class,property" class="bp_form_complete-1580-uri" /> Try it:
       
    • Search only 'property' objects. Use data-bp_objecttypes="property". For example, use this code to search for 'function' among only 'property' objects from the "Adverse Event Reporting Ontology".
      <input type="text" name="d" data-bp_objecttypes="property" class="bp_form_complete-1580-uri" /> Try it:
       
    • Search 'class' and 'individual' objects. Use data-bp_objecttypes="class,individual". For example, use this code to search for 'seizure' among 'class' and 'individual' objects from the "Adverse Event Reporting Ontology".
      <input type="text" name="d" data-bp_objecttypes="class,individual" class="bp_form_complete-1580-uri" /> Try it:
       
    • Only search 'individual' objects. Use data-bp_objecttypes="individual". For example, use this code to this search for 'seizure' only among 'individual' objects from the "Adverse Event Reporting Ontology".
      <input type="text" name="d" data-bp_objecttypes="individual" class="bp_form_complete-1580-uri" /> Try it:
